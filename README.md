# 🔑 Smart School Access Control System
### *High School Senior Graduation Project (Grade 12)*

This project is an **IoT-based smart security solution** designed to modernize access to restricted school areas. It replaces traditional keys with a digital, trackable, and secure RFID system.

## 🚀 Project Overview
The system utilizes an **ESP32** microcontroller integrated with an **RC522 RFID reader** and an **SG90 servo motor**. It features a local web server for real-time monitoring and was fully prototyped using 3D printing technology.

## 🛠️ Technical Stack
*   **Microcontroller:** ESP32 (Dual-core, built-in Wi-Fi)
*   **Security:** RC522 RFID module for high-accuracy authentication
*   **Actuator:** SG90 Servo motor for rapid lock mechanism
*   **Design:** Custom modular enclosure designed in **Blender**
*   **Slicing:** **Bambu Studio** used for print optimization
*   **Manufacturing:** FDM 3D Printing
